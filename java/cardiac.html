JavaScript


INVENTED

By : Netscape Communications Corporation
Date : May to December 1995

Marc Andreessen, founder of Netscape Communications and part of the ex-Mosaic team, had the vision that the web needed a way to become more dynamic. Animations, interaction and other forms of small automation should be part of the web of the future


USED IN AREAS

1. Web Applications (Frontend) - JavaScript is the defacto standard for client side web development.

2. Web Servers (Backend) - With the advent of Node JS, JavaScript is a very popular choice for highly scalable Backend Servers due to its non-blocking nature.

3. Mobile Applications - HTML/JS/CSS can be used to develop mobile applications. Apache Cordova can be used to develop cross platform mobile applications that work with iOS, Android and Windows Mobile devices.

4. Desktop Applications

TYPES OF FRAMEWORK

Javascript Frameworks are rewritten libraries of javascript functions to help developers to solve complex solutions. It provides an excellent structure hence we can create cross-browser compliant code.
1.Relay----Relay is a JavaScript framework for building data-driven React applications
2.jQWidgets---jQWidgets provides a comprehensive solution for building professional web sites and mobile apps. It is built entirely on open standards and technologies like HTML5, CSS, JavaScript and jQuery.
3.jEliJS---Javascript MVC framework for web development.
4.Kickoff---A lightweight front-end framework for creating scalable and performant, responsive sites.
5.Onsen UI--Onsen UI is a library of UI components to help you create beautiful hybrid and mobile web apps for both Android and iOS using Javascript.
6.Cylon.js---Cylon.js is a JavaScript framework for robotics, physical computing, and the Internet of Things. It makes it incredibly easy to command robots and devices.
7.Jest---Jest is a JavaScript unit testing framework, used by Facebook to test services and React applications.
8.Cycle.js---A functional and reactive JavaScript framework for cleaner code.
9.Feathers---A minimalist, real-time JavaScript framework built for NodeJS, React-Native and the browser.
10.R---A lightweight javascript MVVM framework.
11.AVA---Futuristic JavaScript test runner.


LIBRARIES MOST USED

React is a view library that provides an API for rendering your HTML via JavaScript .  It is highly declarative and promotes the reuse of code through the building of components that manage their own data and the state of that data.  It provides you with a terrific model to follow in order to build reactive web interfaces.
JSX provides a mechanism for writing HTML-like syntax that gets converted into JavaScript functions that React can understand.  It gives you a very intuitive and declarative syntax that feels like HTML and provides you with a mechanism for interacting with what React calls the Virtual DOM (an in memory representation of the DOM).
